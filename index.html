lifecycle-map — viewer

interactive swim-lane lifecycle viewer

Render any swim-lane lifecycle map from a JSON or YAML source. Five ways to load:

Try an example →

Load the hiring-pipeline example to explore the schema and interactions.

Try multi-language →

A smaller example with strings in en, pt, and es. Switch language in Settings → Language.

Load from URL → ?src=https://…

Pass a JSON or YAML URL as a query param. Works with public Gists, raw GitHub URLs, or any CORS-enabled endpoint.

Embed in URL → #data=…

Self-contained URLs with gzip-compressed JSON in the fragment. Great for AI agents — no hosting needed.

Or just drag a file in →

Drop a .json or .yaml file anywhere on this page.
